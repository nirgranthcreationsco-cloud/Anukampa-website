# Anukampa Website

Static Hindi-first landing page for **अनुकम्पा**, a Samarpan Sahayog Foundation initiative focused on education and medical support.

## Run locally

Start the local dev server:

```bash
npm run dev
```

Then visit `http://localhost:5173`.

## Source assets

The provided brochure images are stored in `assets/`, including the logo, membership form, donation details, and cropped QR code.
